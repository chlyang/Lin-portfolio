VOICE ASSISTANT

Voice Assistant is a function of the phone system, which allows the users to access services by conversations

Background

Voice Assistant is a pre-installed application in Huawei Phone which can be activated anytime.

Project Goal

This project targets at exploring the framework and design pattern for the Voice Assistant.

Competitor

Although the voice assistant is a relatively new design area, there have been lots of competitive products in the market. On analyzing the existed products, I have summarised some basic interaction patterns.

Industrial Scan

Voice assistant skills classification

According to Huawei’s business strategy, in the future more and more skills will be incorporated into the assistant, which enables the users to better use Huawei devices. Therefore we should choose the pattern that support those skills.

I summarized all of the assistant’s skills and try to figure out what is the best UX pattern for each skill.

Design strategy

A solution that combines quick response and multi-round conversation together

1.Avatar design

2.Interaction Flow

Future

In the future, we will try to incorporate more emotions into the voice assistant, making it a more intelligent assistant.

Summary

Voice assistant is a high-tech project which also requires the designer to learn a bit about machine learning and basic technical knowledge. In the design process, we explore ways to better display the contents and enhance the experience for the user, which will be valuable for future intelligent product design.
