VOICE ASSISTANT

Voice Assistant is a function of the phone system, which allows the users to access services by conversations

Background

Voice Assistant is a pre-installed application in Huawei Phone which can be aroused anytime.

This project targets at exploring the framework and design pattern for the Voice Assistant.

Although voice assistant is a relatively new design area, there have been lots of competitive products in the market. On analysing the existed products, I have summarised some basic interaction patterns.

What are the common patterns for voice assistant?

Take calling taxi as an example, let’s compare the patterns

Prototype A：Conversation

Conversational interaction will be the first interaction mode.

Prototype B：Search

Developed from the traditional search method and focus on graphical interface.

Prototype C：Floating

The users can arouse the assistant anytime.

Pattern comparison

The three patterns have merits and demerits respectively. Pattern 1 is like an introverted friend who can communicate with you quietly in text. Pattern 2 is a more extroverted friend who can constantly give you feedback. And pattern 3 shares the common personalities of extrovert and introvert, which enables it to show up while needed.

Voice assistant skills classification

After analyzing all the current skills of the voice assistant, I figure that the best way to design is combining floating and full screen status together so that the assistant can support both quick response and multi-round conversation.

Hybrid Experience Framework

Now it’s the design stage. On thinking how the floating status and the full screen status can be organically combined together, I separated the question into three basic dimensions.

1.Avatar

Q1 How the assistants interact with the user

Q2 How to define the status and the templates

2.Flow

Q3 What’s the overall interaction flow for voice assistant

Q4 How to enter full-screen mode

3.Ability

Q5 How to transform from different modalities

Future

In the future, we will try to incorporate more emotions to the voice assistant, making it a more intelligent assistant.

Summary

Voice assistant is a high-tech project which also requires the designer to learn a bit about machine learning and some basic technical knowledge.

We explore the ways to better display the contents and interact with the users and gain a lot from that experience.
